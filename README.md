_FRONTEND_ - **VueJS/Vuex/Bootstrap**

_BACKEND_ - **Laravel/Blade**

-----

**Rest API**
-----
* **GET**

    * **/users**
      * Get all users 
    * **/users/{user_id}**
      * Get user 
    * **/users/{user_id}/posts**
      * Get posts user 
    * **/users/{user_id}/posts/{post_id}**
      * Get post user
    * **/users/{user_id}/followers**
      * Get followers user 
    * **/users/{user_id}/follows**
      * Get follows user 
    * **/posts**
      * Get posts 
    * **/posts/{post_id}**
      * Get post 
    * **/posts/{post_id}/comments**
      * Get post comments
    * **/posts/{post_id}/reactions**
      * Get post reactions
